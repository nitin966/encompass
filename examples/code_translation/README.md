# Code Translation

Java-to-Python translation with EnCompass search.

```
simple_experiment/     # single branchpoint, greedy
complex_experiment/    # 5 branchpoints, 6 search strategies
```

## Quick Start

```bash
# simple
python examples/code_translation/simple_experiment/baseline_vanilla.py
python examples/code_translation/simple_experiment/run_experiment.py

# complex  
python examples/code_translation/complex_experiment/run_all_strategies.py
```
